
# 🛠️ VisionFerre: E-Commerce Inteligente con IA
VisionFerre es una ferretería en línea que resuelve el problema de identificar piezas técnicas (tornillería, abrazaderas, anclajes) mediante una imagen, utilizando Inteligencia Artificial.

## 🚀 Características Principales
Reconocimiento de Imágenes (IA): Los usuarios pueden subir una foto de una pieza física y el sistema utiliza Amazon Rekognition (Custom Labels) para identificarla y mostrar productos coincidentes en el inventario.

Asistente Inteligente (IA): un chatbot brinda detalles de las especificaciones de los productos en tiempo real con Amazon Bedrock.
